# GitHub Features for Tracking Tasks

## Labels

| Label | Meaning |
|---|---|
| todo | Work has not started |
| in-progress | Work has started but is not finished |
| done | Work is finished and checked |
| user-story | Main user story |
| task | Small task under a user story |
| documentation | Report or diagram work |
| code-review | Pull request review work |
