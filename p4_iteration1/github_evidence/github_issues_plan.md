# GitHub Issues and Labels Plan

## Labels

| Label | Meaning |
|---|---|
| todo | Work has not started yet |
| in-progress | Work has started but is not finished |
| done | Work is completed and checked |
| user-story | Main user story item |
| task | Smaller task under a user story |
| documentation | Diagram, report, or planning work |
| code-review | Pull request review work |
